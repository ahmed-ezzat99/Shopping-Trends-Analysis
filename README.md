# Shopping Trends Analysis
Python & Power BI Project

## Project Objective
The objective of this project is to analyze customer shopping behavior and sales performance in order to identify revenue drivers, customer segments, and actionable business insights.

This project demonstrates practical skills required for Data Analyst and BI Analyst roles, including data exploration, KPI analysis, and dashboard development.

---

## Project Scope
- Analyze customer demographics and purchasing behavior
- Identify top-performing product categories and items
- Evaluate sales performance by region
- Assess the impact of shipping and payment methods on revenue
- Present insights through an interactive Power BI dashboard

---

## Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Power BI (Data Modeling, DAX, Visualization)
- Jupyter Notebook

---

## Project Deliverables
- Cleaned and analyzed dataset
- Exploratory Data Analysis using Python
- Interactive Power BI dashboard
- Business-focused insights and recommendations
