# Analysis Methodology

## 1. Data Preparation
- Reviewed dataset structure and data types
- Cleaned categorical values
- Validated numerical fields

## 2. Exploratory Data Analysis
- Distribution analysis for sales and customers
- Category and product-level aggregation
- Demographic segmentation

## 3. Power BI Modeling
- Created relationships and calculated measures
- Built KPIs for sales, customers, and revenue contribution
- Designed dashboard focused on business interpretation

## 4. Insight Generation
- Identified performance drivers
- Translated findings into actionable business recommendations
